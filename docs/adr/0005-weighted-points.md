# ADR 0005: 重みポイントによる負荷バランス評価

## ステータス
承認済み

## コンテキスト
家事・育児の負荷バランスを評価する指標を決定する必要があった。
単純な時間集計では、難易度・精神的負担が異なる家事を公平に評価できない。

## 決定
各カテゴリに重みポイントを設定し、「重みポイント × 実施回数（または時間）」で負荷バランスを算出する。
重みは家族が話し合ってアプリ内で設定する。

## 理由
- 時間だけでは家事の質・難易度の差が反映されない
- 家族が合意した重みを使うことで、評価に納得感が生まれる
- 初期値を1とすることで、重み未設定でも時間ベースとして機能する

## 結果
- `task_category` に `weight`（int, default: 1）を追加する
- `load_balance` に `weighted_points` を追加し、負荷バランスの計算に使用する
- 重みの設定はUC6（カテゴリの重みを設定する）で行う
