# ADR 0001: Spring Boot の採用

## ステータス
承認済み

## コンテキスト
家庭内の家事・育児を記録・集計・振り返るWebアプリケーションを開発する。
バックエンドのフレームワークを選定する必要があった。

## 決定
Spring Boot 3.x を採用する。

## 理由
- Java/Springの豊富なエコシステムとドキュメントが揃っている
- Spring Data JPAによりリポジトリ実装のコストが低い
- Spring Schedulerによる週次バッチ処理を追加ライブラリなしで実現できる
- DDDライクなレイヤー構成との親和性が高い

## 結果
- Java 21 + Spring Boot 3.3.5 を使用する
- ビルドツールはMavenを使用する
