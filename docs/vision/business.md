# ビジネスビジョン

## 対象ユーザー

家庭内で家事・育児を分担している家族（夫婦・パートナー等）。

## プロダクト概要

家事・育児の記録・集計・振り返りを支援するWebアプリケーション。  
誰が何をどれだけやったかを可視化し、家族が事実に基づいて対話できる場を提供する。

## 提供する価値

| 価値 | 説明 |
|------|------|
| 見える化 | 日々の家事・育児の貢献が数値とグラフで確認できる |
| 公平な評価 | 時間だけでなく重みポイントで各家事の負担を適切に評価できる |
| 対話の質向上 | 感覚ではなく事実をもとに「ありがとう」や「もう少し助けて」が言える |
| 継続的改善 | 振り返りと改善アクションのサイクルで家庭運営が少しずつ良くなる |
