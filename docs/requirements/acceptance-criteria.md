# 受け入れ条件

## UC1: 家事・育児を記録する

- [ ] カテゴリ・担当者・所要時間を選択・入力して記録を保存できる
- [ ] 実施日時は記録操作時点で自動設定される（ユーザー入力不要）
- [ ] 保存後、記録完了の旨がユーザーに伝わる

---

## UC2: 週次実績を確認する

- [ ] 週ごとの家事育児記録を集計して表示できる
- [ ] 誰が・何の家事を・何時間やったかを一覧で確認できる
- [ ] デフォルトは円グラフで表示される
- [ ] グラフ種別を複数の中から切り替えられる
- [ ] 確認する週を指定できる

---

## UC3: 負荷バランスを確認する

- [ ] UC2と同一画面内で負荷バランスを切り替えて表示できる
- [ ] 家族メンバーごとの重みポイント合計が表示される
- [ ] 家族メンバーごとの負担割合（%）が表示される

---

## UC4: 改善アクションを記録する

- [ ] 振り返り画面でアクション内容と担当者を入力して登録できる
- [ ] 登録されたアクションが一覧で表示される
- [ ] アクションの実践状況を「未着手」「実践中」「完了」で更新できる

---

## UC5: 振り返りを比較する

- [ ] 今回と前回の重みポイント合計を並べて比較できる
- [ ] 今回と前回の負荷バランス（割合）を並べて比較できる
- [ ] 前回の改善アクションとその達成状況が確認できる
- [ ] 初回（前回の振り返りが存在しない場合）は今回分のみ表示される

---

## UC6: カテゴリの重みを設定する

- [ ] カテゴリ一覧が表示され、各カテゴリに重みポイントを設定できる
- [ ] 重みは1以上の整数で設定できる（未設定時のデフォルトは1）
- [ ] 設定した重みが負荷バランスの計算にすぐ反映される
