# ユーザーストーリー

## Epic

家庭内の負荷を可視化し、より良い役割分担を実現したい

---

## User Story 1

家族メンバーとして、家事・育児を記録したい。

なぜなら、誰が何をどれだけ実施したかを振り返りたいからだ。

## User Story 2

家族メンバーとして、週ごとの家事・育児の実績を確認したい。

なぜなら、感覚ではなく事実に基づいて会話したいからだ。

> このアプリのコアとなるストーリーです。

## User Story 3

家族メンバーとして、負荷の偏りを確認したい。

なぜなら、不公平感を減らしたいからだ。

## User Story 4

家族メンバーとして、改善アクションを記録したい。

なぜなら、前回決めたことが実践できたか確認したいからだ。

## User Story 5

家族メンバーとして、前回の振り返り結果と今回の結果を比較したい。

なぜなら、家庭運営が改善しているか確認したいからだ。
