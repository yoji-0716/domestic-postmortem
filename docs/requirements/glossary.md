# 用語集

| 用語 | 説明 |
|------|------|
| 家族メンバー | アプリを利用する家庭の構成員。全員が同等の権限を持つ。初期データとして固定登録。 |
| 家事育児カテゴリ | 家事・育児の種別（例：皿洗い、洗濯、保育園の送迎）。初期データとして固定投入。 |
| 重みポイント | 各カテゴリに設定する負荷の評価値。家族が話し合って設定する。 |
| 家事育児記録 | 家族メンバーが実施した家事・育児の1件の記録。カテゴリ・担当者・所要時間を持つ。 |
| 週次実績 | 1週間分の家事育児記録を集計したもの。毎週自動生成される。 |
| 負荷バランス | 週次実績をもとに算出した、家族メンバーごとの重みポイント合計と割合。 |
| 振り返り | 週次実績・負荷バランスを参照して行う定期的な家族会議。今回と前回を比較できる。 |
| 改善アクション | 振り返りで決定した具体的な行動計画。担当者・内容・実践状況を持つ。 |
| 実践状況 | 改善アクションの進捗。「未着手」「実践中」「完了」の3段階。 |
